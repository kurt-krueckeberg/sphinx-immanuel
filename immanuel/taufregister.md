# Taufregister (Baptism Register)

**Taufregister** = **baptism register**. The baptism records follow a fixed format and use a fixed vocabulary with
the definitions given below:

```{list-table}
:header-rows: 1

* - Word
  - Meaning
* - Taufregister
  - baptism register
* - geboren
  - born
* - Eltern
  - parents
* - geb.
  - abbreviation "born (as)", i.e., maiden name
* - Pathen
  - baptismal sponsor, godparents
* - Siehe
  - see
* - Seite
  - page
* - Januar
  - January
* - Februar
  - February
* - März
  - March
* - Mai
  - May
* - Juni
  - June
* - Juli
  - July
* - Oktober
  - October
* - Dezember
  - December
```
